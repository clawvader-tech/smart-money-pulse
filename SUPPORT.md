## Troubleshooting

### "NANSEN_API_KEY not set"

```bash
export NANSEN_API_KEY="your_api_key_here"
```

### "Command not found: nansen"

Install the Nansen CLI according to their documentation.

### "No data returned"

- Check your Nansen API key is valid
- Verify you have credits remaining
- Try a different timeframe (24h, 7d, etc.)
