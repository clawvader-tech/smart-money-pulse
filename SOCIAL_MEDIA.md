Blockchain: Base L2 (Ethereum security, low fees)

---

3/6 Distributed Training

We don't train one giant model.

Instead:
• Base model: Qwen2.5-2B (consumer-friendly)
• Contributors train LoRA adapters
• Adapters merge into improved model
• Quality verified by consensus

Federated learning with crypto incentives.

4/6 Inference Network

Users submit queries → Coordinator assigns to GPU agents → Agents return results → Consensus verifies → Rewards distributed

All in <500ms.

vLLM enables batching for efficiency.
