Features

Ranked trading opportunities with confidence scores, flow magnitudes, and smart money status. Updated continuously.

MCP Server

Model Context Protocol integration — connect any AI agent (Claude, GPT, Gemini) to Atlas for autonomous trading decisions.

💳

Native crypto micropayments. Agents pay per API call. No accounts, no subscriptions — usage-based pricing.

World-class HTML dashboard with signal feed, trade history, positions, system health, and OKX wallet widget.

Quick Start

Up and running in 60 seconds

One command to install. One command to run.

# 1. Clone the repo
git clone https://github.com/your-username/smart-money-pulse.git
cd smart-money-pulse

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set your API keys
cp .env.example .env
# Edit .env with your NANSEN_API_KEY

# 4. Run!
python smart_money_pulse.py --chain base --top-n 5
