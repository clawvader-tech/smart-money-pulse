Features

Everything you need to track smart money

From real-time CLI signals to AI agent integration — built for traders and developers who need an edge.

📈

Real-Time Signals

Ranked trading opportunities with confidence scores, flow magnitudes, and smart money status. Updated continuously.

🤖

MCP Server

Model Context Protocol integration — connect any AI agent (Claude, GPT, Gemini) to Atlas for autonomous trading decisions.

💳

x402 Payments

Native crypto micropayments. Agents pay per API call. No accounts, no subscriptions — just usage-based pricing.

💻

CEO Dashboard

World-class HTML dashboard with signal feed, trade history, positions, system health, and OKX wallet widget.

🔗

Multi-Chain

Base, Ethereum, Solana, Arbitrum, and Optimism. All major L2s and DeFi chains supported.

🔒

Secure by Design

No API keys in code. Environment-based config. OKX wallet integration with local signing. Your keys stay yours.
