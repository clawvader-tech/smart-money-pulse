Features

Everything you need to track smart money

From real-time CLI signals to AI agent integration — built for traders and developers who need an edge.

📈

Real-Time Signals

Ranked trading opportunities with confidence scores, flow magnitudes, and smart money status. Updated every 5 minutes.

🤖

MCP Server

Model Context Protocol integration — connect any AI agent (Claude, GPT, Gemini, Codex) for autonomous trading.

💳

x402 Payments

Native crypto micropayments. Agents pay per API call. No accounts, no subscriptions.

💻

CEO Dashboard

World-class HTML dashboard — signal feed, trade history, positions, system health, wallet widget.

🔗

Multi-Chain

Base, Ethereum, Solana, Arbitrum, and Optimism. All major L2s supported.

🔒

Secure by Design

No API keys in code. Environment config only. OKX wallet with local signing.

Quick Start

Up and running in 60 seconds

One command to install. One command to run.

1

Clone the repo

git clone https://github.com/clawvader-tech/smart-money-pulse.git

2

Install dependencies

pip install -r requirements.txt

3

Configure

cp env.example .env # add your NANSEN_API_KEY

4

Run!

python smart_money_pulse.py --chain base --top-n 10
