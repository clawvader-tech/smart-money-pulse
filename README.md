# Smart Money Pulse Monitor

Track whale wallets. Spot accumulation. Trade with conviction.

## Quick Start
```bash
git clone https://github.com/clawvader-tech/smart-money-pulse.git
cd smart-money-pulse
pip install -r requirements.txt
export NANSEN_API_KEY=your_key
python smart_money_pulse.py --chain base --top-n 10
```

## Features
- Real-time trading signals with confidence scores
- MCP Server for AI agent integration
- x402 payment gateway for crypto micropayments
- CEO Dashboard (open dashboard.html in browser)
- Multi-chain: Base, Ethereum, Solana, Arbitrum, Optimism

## Files
- smart_money_pulse.py — Main CLI tool
- mcp_server.py — AI agent MCP integration
- payment_gateway.py — x402 payment server
- dashboard.html — CEO dashboard
